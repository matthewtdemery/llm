## What makes this MVP "finished" for the Monday call

- Polished product-facing landing section.
- Persistent taste profile using browser localStorage.
- Fast profile presets: Tom classic night, date night, family safe, theater energy, award winners, under 2 hours.
- Search across title, mood, genre, audience, provider, synopsis, and quality signals.
- Audience, mood, genre, route, decade, runtime, and confidence filters.
- Trailer-first decision cards.
- Details modal with route explanation and MVP boundary note.
- Watchlist modal.
- Save / not-for-me / trailer-open feedback capture.
- User-test form for tester notes and usefulness score.
- Exportable JSON session receipt.
- Print brief button.
- Demo profile button for the call.
- Manifest file for app-like packaging later.

## Intentional boundaries

- Demo data only.
- No live streaming API.
- No live theater showtimes.
- No backend.
- No accounts.
- No payment/ticketing.
- No provider contracts.

The MVP is designed to validate the wedge before funding live data integrations or native app development.
