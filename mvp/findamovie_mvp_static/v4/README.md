## Base

This version was created as a separate `v4` from the existing `v1` webroot copy, then the handwritten notes were applied as a new flow.

## Added from sketches

- Home page with three easy steps.
- Local demo profile name/passcode status.
- Search profile: genres, release range, custom year range, ad-free/sponsored/VOD/all.
- Available platforms: theater ZIP, AMC, Regal, Alamo, Phoenix, All theaters, and streaming providers.
- Titles + trailers section using trailer-first cards.
- Where-to-watch modal with Tribute showtimes and IMDb title links.
- Save or drag movie cards to My Library.
- Export profile/session JSON.

## Boundary

This is a static MVP. Demo passcode is not authentication. Availability and links are route prototypes only; production needs backend account/security, live metadata, provider availability, showtimes, and ticketing integrations.
